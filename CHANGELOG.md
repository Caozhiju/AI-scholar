# Changelog

## LingAI Scholar v1.0.0 (2026-06-11)

### Phase 1 — Python 科研基础 ✅
10 门课程覆盖 Python 文件处理、文本清洗、词频统计、共现分析、可视化与科研自动化。

### Phase 2 — 计算语言学 AI ✅
15 门课程覆盖中文分词、词性标注、命名实体识别、依存句法、文本分类、情感分析与偏误检测。

### Phase 3 — 教育 NLP ✅
15 门课程覆盖可读性分析、学习者语言分析、题型自动生成、HSK 难度标注、ChatGPT 辅助评估等教育 NLP 核心主题。

### Phase 4 — 大模型 LLM ✅
23 门课程覆盖 LLM 原理、Prompt 工程、RAG、Agent 设计、模型评测、模型微调与安全对齐。

### 审计系统 ✅
